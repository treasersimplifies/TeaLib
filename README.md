# TeaLib

Feel free to use these files for learning or demo usage.

- `staticWebPageTemplate` : An all-in-one static(without using JavaScripts) web project for learning and demo usage.
- `ecmaLearn.js` : An all-in-one EcmaScripts file for learning and demo usage.
- `DomBomWebPageTemplate` : An all-in-one web project for showing DOM and BOM regular usage.
- `MarkDownTemplate.md` : An all-in-one markdown project(cloned from MacDown) for learning and demo usage.
